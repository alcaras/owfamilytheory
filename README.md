# owfamilytheory

A single standalone page on **Old World family-class theory** — how to play each
nation's fixed family classes.

Open **`index.html`** in a browser (no build step). One card per nation lists every
valid combo, generated from the rules: field three families — one **cap**, one
**mil seat**, a **third** — and feed at most one **spammer**. The page's
`genCombos()` is those rules made executable.

Crests and family-class icons in `img/` are extracted from the Old World install
via the [owreference](https://github.com/alcaras/owreference) pipeline.
